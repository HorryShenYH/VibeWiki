# Knowledge

Reviewed facts, explanations, and project memory that are useful but not
necessarily executable skills.

## VibeWiki Product Positioning

Source: VibeWiki self-review conversation, 2026-06-19.

VibeWiki's strongest product idea is to act as a trusted memory compiler for AI
conversations:

```text
AI conversation -> evidence -> candidate memory -> human review -> approved memory -> reuse
```

The important distinction is that one conversation should not automatically
become one skill. A conversation can contain facts, issues, todos, ideas,
research notes, directions, prompt patterns, skilllets, and workflows. Different
conversations may improve the same reusable unit over time.

Chinese positioning:

> VibeWiki 把人与 AI 协作中产生的经验、灵感、问题和技能，编译成可审查、可演化、可复用的项目记忆。
