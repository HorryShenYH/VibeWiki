# Venus Examples

Venus is the first serious VibeWiki case study because it has exactly the kind
of project memory that AI coding agents tend to lose: simulator commands,
instruction semantics, RTL/VEMU alignment, benchmark configuration, and
fixed-point numerical caveats.

## Examples

- `sample_session.md`: small synthetic starter example
- `real_sessions/cmxmul_ofdm/`: real session about `VCMXMUL`, OFDM fixed-point
  error, and why the explicit multiply path remains the default
